# Telegram Бот по Нумерологии

Этот бот рассчитывает нумерологический расклад (Число Жизненного Пути и Квадрат Пифагора) по дате рождения.

## Основные функции
- **Проверка подписки**: Доступ к функциям только после подписки на ваш Telegram-канал.
- **Сохранение данных**: Бот запоминает пользователей и их даты рождения.
- **Личный профиль**: Команда `/profile` позволяет увидеть свои данные и актуальный расклад.
- **Нумерологический расчет**: Автоматический расчет по введенной дате.

## Установка и запуск

1. **Клонируйте репозиторий или скопируйте файлы.**
2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Настройте переменные окружения:**
   Создайте файл `.env` в корневой папке:
   ```env
   TELEGRAM_TOKEN=ваш_токен_от_BotFather
   CHANNEL_ID=@ваш_канал
   CHANNEL_URL=https://t.me/ваш_канал
   ```
   *Или отредактируйте `config.py` напрямую.*

4. **Запустите бота:**
   ```bash
   python bot.py
   ```

## Дальнейшее развитие
В проект заложена база для добавления платной подписки на библиотеку практик. Вы можете расширить модель `User` в `database.py`, добавив поле `is_premium`, и реализовать интеграцию с Telegram Payments.
